# Validation

The Validation endpoint allows you to check subtitle files for common structural errors before processing them. This is useful for ensuring file integrity.

## Check Subtitles

This endpoint parses the file and reports any syntax errors, overlapping timestamps, or encoding issues.

```bash {{ title: 'Example Request' }}
curl -X POST https://subtitlecleaner.chidozienwoga.workers.dev/api/v1/validate \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "file=@movie.srt"
```
